# JavaScript Arrays

1. Defining Arrays
  const colors = ['red', 'blue', 'white'];
  const arr = [10, 20, 'three', [10, 20, 30], null];

2. Accessing Array Items
  arr[0] === 10
  arr[2] === 'three'
  arr[3][0] === 10

  arr[0] = 99;
  arr === [99, 20, 'three', [10, 20, 30], null];

3. Arrays Length
  arr.length === 5;
  colors.length === 3;
  colors[colors.length - 1]; // last item in the array
