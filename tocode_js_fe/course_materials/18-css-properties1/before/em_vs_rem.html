Em demo

Em is used to describe font sizes relative to their container element. Many people use Em because it allows us to change ALL text sizes in the page by changing a single CSS value

However the problem with Em is that they're hard to reason about

If .inner div has a size in em, then my size will be relative to .outer font size

rem demo

rem (Relative Em) is used to describe font sizes relative to their container element. Many people use Em because it allows us to change ALL text sizes in the page by changing a single CSS value

We love rem because they're ALWAYS relative to the font-size in html element

If .inner div has a size in rem, then my size will stil be relative to html's font size
